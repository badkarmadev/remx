# Remx.css Documentation

Welcome to the Remx.css documentation.

## Installation

```bash
npm install remx.css
# or
yarn add remx.css
```

## Usage

### Full Bundle

Import the complete framework:

```javascript
import 'remx.css';
```

### Modular Imports

You can import specific modules to keep your bundle size low:

```javascript
import 'remx.css/reset';
import 'remx.css/layout';
import 'remx.css/typography';
// ... and so on
```
