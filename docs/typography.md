# Typography

Utilities for controlling font size, weight, alignment, and other text properties.

## Font Size

Sets the `font-size` property.

| Class | Variable | Description |
| :--- | :--- | :--- |
| `.fs-xxs` | `--font-size-xxs` | Extra extra small text |
| `.fs-xs` | `--font-size-xs` | Extra small text |
| `.fs-s` | `--font-size-s` | Small text |
| `.fs-m` | `--font-size-m` | Base text size |
| `.fs-l` | `--font-size-l` | Large text |
| `.fs-xl` | `--font-size-xl` | Extra large text |
| `.fs-xxl` | `--font-size-xxl` | 2x large text |
| `.fs-3xl` | `--font-size-3xl` | 3x large text |
| `.fs-4xl` | `--font-size-4xl` | 4x large text |
| `.fs-5xl` | `--font-size-5xl` | 5x large text |
